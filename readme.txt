== Installation ==

1. Upload the entire 'simple-aweber-integration' folder to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the settings under Settings -> Simple Aweber Integration in your WordPress admin panel.

For basic usage, you can also have a look at the [http://www.schurpf.com/simple-aweber-integration/](http://www.schurpf.com/simple-aweber-integration/). To find help, pls read [http://www.schurpf.com/simple-aweber-integration-help/](http://www.schurpf.com/simple-aweber-integration-help/).
